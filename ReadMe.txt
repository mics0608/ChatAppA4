Open VSCode copy or drag the folder 'ChatAppA4' to Explorer

Run Server (Backend)
1. Open terminal in VSCode, cd to backend.
2. Type npm run start.

Run Frontend
1. Split the terminal in VSCode, cd to frontend.
2. Type npm run dev.
3. Frontend will run, locate the link that says the 'Local', copy that link to a browser.
4. Create a username and click enter.
5. New usernames can be created or run in different tabs.
